# ClipperInstantReplay

U-22プログラミングコンテスト2025 応募作品  
サービス紹介URL : https://protopedia.net/prototype/private/95883d9b-7c61-47f9-9bed-05a82f640b8d

## このプロジェクトについて

本プロジェクトは、プログラミング学習を開始して最初に取り組んだチーム制作作品です。  
デスクトップの操作を遡って録画・保存を行いその場でカット編集ができるインスタントリプレイツールを開発しました。

## 開発背景

U-22プログラミングコンテスト2025への応募にあたり、作品概要をProtoPediaに集約しています。  
開発背景やデモ動画については上記URLをご参照ください。

## 主な機能

- **バックグラウンド録画**: 常に最新の操作画面を一時保存。
- **インスタントリプレイ**: 任意のタイミングで過去の映像を保存。
- **カット編集**: 保存した動画の必要な部分だけをその場で抽出。

1. [Releases](https://github.com/あなたのユーザー名/リポジトリ名/releases) から最新版をダウンロードしてください。

3. `ClipperInstantReplay.exe` をダブルクリックして実行してください。
